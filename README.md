Opamok은 오목대결을 할때
- 최적화 된 수와 전략, 
- 많은 수순 계산
- 빠른 생각

이렇게 대결합니다.

# 전략

ai의 전략은
1. 중앙 십자 빌드 - 중앙에 두고 십자 모양으로 확장 - 공격과 수비 균형 
2. 대각선 빌드 - 중앙에서 대각선으로 빠르게 연결 - 오목 완성 속도가 빠름 
3. 이중 삼 빌드 - 삼을 두 군데 동시에 만들어 압박 - 상대가 막아도 다른 길 열림
4. 4-3 공격법 - 한 수로 4목과 3목을 동시에 만드는 전술 - 상대가 막을 수 없는 필승 루트 
5. X진법 - 중앙에 X자 형태로 돌을 배치해 대각선 확장 - 빠른 대각선 공격 가능 
6. 날일진법 - ‘一’ 모양으로 돌을 배치해 공격·수비 병행 - 후공(백돌)에게 유리 
7. 포석 빌드 - 초반에 돌을 넓게 퍼뜨려 여러 가능성 확보 - 다양한 공격 루트 준비 
8. 함정 빌드 - 일부러 상대가 막게 유도하고 다른 쪽에서 오목 완성 - 심리전 활용 
9. 속도전 빌드 - 초반에 삼삼을 빠르게 만들어 상대가 수비에 몰두 - 빠른 압박으로 승률 상승

이런 빌드를 이용합니다.
